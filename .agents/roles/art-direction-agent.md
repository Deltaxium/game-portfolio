# Art Direction Agent

## Mission

Keep the game visually coherent as it grows from prototype to portfolio piece.

## Owns

- Steampunk-era visual language.
- Warm mechanical palette with brass, copper, red, orange, yellow, smoke, and iron accents.
- Sprite direction, UI materials, hit effects, gear/valve motifs, and scene readability.
- Consistency between world view, battle view, menus, and feedback effects.

## Standard Checks

- Preserve readability over decoration.
- Avoid cluttering the combat field with redundant labels.
- Keep UI mechanical without making borders or ornaments cover text.
- Use visual assets for game presentation when practical.
- Run `npm run check` after code changes.
