# Stormwake Isles

A top-down real-time Phaser roguelike about piloting a modular airship through floating sky islands, storms, pirates, ruins, and ancient machines.

## Core Loop

Start a run, choose a route, fly to a sky island, fight or explore, loot temporary upgrades, survive elite encounters, then spend permanent currency after victory or defeat.

## Current Prototype Scope

- Phaser + Vite browser game scaffold.
- Direct airship control with boost and forward cannons.
- Biome map catalog for every planned region.
- Data-driven biome palettes, hazards, factions, and encounter tags.
- Weathered industrial sky-fantasy art direction using brass, patched wood, oxidized steel, canvas, cloud glass, and ancient glow accents.

## Commands

```bash
npm install
npm run dev
npm run build
```
